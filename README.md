# FirstVision AI

FirstVision AI is a real-time object detection web application that uses your webcam to identify objects and instantly search them on the web.

Built using:
- Frontend: HTML, CSS, JavaScript
- Backend: Flask (Python)
- AI Model: Hugging Face Inference API

---

## 🔥 Features

✅ Real-time webcam capture  
✅ Object detection using AI  
✅ Instant web search for detected objects  
✅ Clean and simple UI  
✅ End-to-end full-stack project  

---

## 🧠 How It Works

1. The webcam captures an image
2. Image is sent to Flask backend
3. Backend calls Hugging Face model
4. Model returns detected objects
5. Results are displayed with search buttons

---

## 📸 Demo Flow

- Click **Scan**
- System captures frame
- AI detects objects
- Click **Search** to explore results

---

## 🛠️ Tech Stack

| Layer      | Tech                     |
|------------|--------------------------|
| Frontend   | HTML, CSS, JavaScript    |
| Backend    | Flask (Python)           |
| AI Model   | Hugging Face API         |
| Camera     | WebRTC (getUserMedia)    |
